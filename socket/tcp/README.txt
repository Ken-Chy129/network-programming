程序的编译：
# 1．Telnet到linux上，用vi编辑c源程序，并编译运行。（或者把文件下载到本地编辑，编辑后用ftp上传。ftp用户名和密码同上）

1. 编译命令：gcc -o 可执行文件名 c源文件名 -Wall（打开所有警告）
2．编辑、编译并调试客户端程序
3．编辑、编译并调试服务器端程序
客户端程序和服务器端程序都可能有错误，根据编译结果修改。
# 注意：这两个程序的端口使用你学号的最后4位，保证每个人使用的不一样；客户端程序和服务器端程序使用相同端口。

注意：端口需要使用>1024的值，否则需要以root用户身份才能运行。

运行：
1．在shell命令行窗口中运行服务器端程序，命令行为：
./server
2．在另一个shell命令行窗口中运行客户端程序，命令行为：
./client 127.0.0.1
